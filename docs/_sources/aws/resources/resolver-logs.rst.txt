.. _aws.resolver-logs.filters.is-associated:

is-associated
+++++++++++++
Checks LogConfig Associations for VPCs.

:example:

.. code-block:: yaml
        policies:
            - name: r53-resolver-query-log-config-associations
              resource: resolver-logs
              filters:
               - type: is-associated
                 vpcid: "vpc-12345678"

.. _aws.resolver-logs.actions.associate-vpc:

associate-vpc
+++++++++++++
Associates ResolverQueryLogConfig to a VPC

:example:

.. code-block:: yaml

    policies:
      - name: r53-resolver-query-log-config-associate
        resource: resolver-logs
        filters:
          - type: value
            key: 'Name'
            op: eq
            value: "Test-rqlc"
        actions:
          - type: associate-vpc
            vpcid: all
