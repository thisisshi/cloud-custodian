.. _aws.cloudsearch.actions.enable-https:

enable-https
++++++++++++
Enable HTTPs to cloudsearch

:example:

.. code-block:: yaml

        policies:
          - name: enable-https
            resource: cloudsearch
            filters:
              - type: domain-options
                key: Options.EnforceHTTPS
                value: false
            actions:
              - type: enable-https
                tls-policy: Policy-Min-TLS-1-0-2019-07
