.. _tencentcloud.ami.filters.unused:

unused
++++++
Filters images based on usage

true: image has no instances spawned from it
false: image has instances spawned from it

:example:

.. code-block:: yaml

        policies:
          - name: ami-unused
            resource: tencentcloud.ami
            filters:
              - type: unused
                value: true
